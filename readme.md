# Proyecto Examen Final de la Materia de Matemática Discreta (Parte del BackEnd)

En este repositorio se encuentra un proyecto de la clase de Matemática Discreta.

## ¿En qué consiste este proyecto?

Diseñar una aplicación que gestione los permisos de los usuarios en una empresa utilizando relaciones de orden parcial y conjuntos. Los estudiantes pueden programar un módulo donde se visualice si un usuario tiene acceso a un recurso por transitividad (si el Administrador tiene acceso a todo, y el Supervisor hereda del Administrador, etc.).
